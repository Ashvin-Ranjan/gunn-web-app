You now can add Zoom links to each of your classes!

  • You can now specify a link for each period.
  • There's an option to automatically open the link in an embed before a period starts. This can be used to automatically launch Zoom.
  • There's an additional option for opening the link in a new tab, but you first need to enable pop-ups for this website.

There's also a few smaller changes.

  • (Classic) Material design inputs are now used more.
  • Fixed clicking on assignments in periods when swiping is enabled.
